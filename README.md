## Встановлення
Для Linux
1. Клонуйте репозиторій:
    ```
    git clone https://github.com/Trenerkok/Petrucha/tree/Phenix_2-0/phenix/main
    cd your-project
    ```
2. Встановіть залежності (приклад для Python):
    ```
    pip install -r requirements.txt
    ```
Для Windows
1. Завантажте Zip і розпакуйте у зручне місце для вас
2. Завантажте всі залежності перебуваючі у папці з проєктом
    ```
    pip install -r requirements.txt
    ```
## Функції

- Опис основних можливостей і задач, які вирішує проєкт:
    -Наш голосовий помічник може взаємодіяти з компютером сервером і IoT пристроями тобто ардуіно 
     ще він може давати відповіді на запитання по типу яка погода котра година і тому подібне якщо
     він чогось не знає то в режимі джеміні звертається до ШІ і вже вона генерує відповідь на запитання 
- За потреби – додайте приклади використання.

## Додаткові дії для запуску

1. **Налаштування змінних середовища** (якщо потрібно):
    - Створіть файл `.env` на основі `.env.example` та вкажіть актуальні значення.
2. **Інсталяція Vosk**:
    Істалюйте Vosk з https://alphacephei.com/vosk/models/vosk-model-uk-v3.zip
3. **Запуск проєкту**:
   перейдіть до розташуваня ГА і виконайте команду щоб запустити Петручу
    ```
    python Phenix.py     # або відповідна команда для запуску
    ```
